# File Layout Details (`./ctx/docs/code/layouts/files/`)

Path: `./ctx/docs/code/layouts/files/AGENTS.md`

## Purpose

The `ctx/docs/code/layouts/files/` directory contains declarative refinements of file-system layout rules introduced at the parent `layouts/` level. Documents here specify structure for major repository branches and their internal organization.

## Level Map

- `AGENTS.md` — this document, defining the boundaries and structure of the `files/` level.
- `src.md` — structural rules for the implementation layout inside the `src/` directory.

## Level Boundary

This level refines file-layout structure only. It must not define runtime behavior, testing semantics beyond placement structure, or package metadata outside file-system organization.

## Summary

`ctx/docs/code/layouts/files/AGENTS.md` defines the navigation and scope of detailed file-layout documentation.
